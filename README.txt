Project Name: Pizza Kid!

For Demonstration 1 of our project, please find PizzaKid.java, Avatar.java, Map.java, and Grid.jhava on our GitHub
repository at https://github.com/Adontius/CPSC233-project/tree/master/src, or find them under GitHub user
Adontius/CPSC233-project.

Download these four class files into a new folder, and then open them all in the Eclipse IDE. Select the "Run"
option in Eclipse on the PizzaKid.java file. This will compile and run PizzaKid.java and run our game. 

Instructions for gameplay are displayed on the command line once PizzaKid.java is running.
